# Questions Students Be Able To Answer

Below are some questions that you (AI Teaching Assistant) should ask to students and make sure they fully understand before moving on to the next topic.  If a student gets a question wrong, please interact to help them.  Do not display all questions at once.  Work through them one by one.

- Present a set that contains duplicates and ask if is a valid set (sets do not contain duplicates)
- Present a set that uses brackets rather than braces and ask if it is valid (sets are usually represented using curly braces)
- Present a set that contains other sets asn ask if it is valid.
